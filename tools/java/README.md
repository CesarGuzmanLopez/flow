# Portable Java runtimes

This folder holds portable JREs downloaded by `scripts/download_java_runtimes.sh`:

- jre8/ -> Java 8 runtime for AMBIT SyntheticAccessibilityCli.jar
- jre21/ -> Java 21 runtime for EPA WebTEST.jar

These binaries are ignored by Git. Regenerate them anytime by running the download script.
